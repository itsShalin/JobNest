# 🍫 JobNest - Full Stack Job Portal

JobNest is a full-stack job portal built using the MERN stack. The platform allows companies to post and manage job opportunities while enabling users to search for jobs, apply online, and manage their applications through a responsive and user-friendly interface.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/itsShalin/JobNest.git
```

---

## Run The Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## Run The Backend

```bash
cd backend
npm install
npm start
```

---

# 🔐 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```
